1 This repository provides a simple template that can be used to develop ATtiny85
2 based projects. As well as support for a serial bootloader (via the
3 [Microboot](https://github.com/thegaragelab/microboot) software and tools) and
4 template [Fritzing](http://fritzing.org/home/) projects for a basic development
5 board it provides source for a library of commonly used routines. These pages
6 document that source.
7 
8 Some notable features of the library are noted here. Please browse through the
9 full documentation to see other features.
10 
11 # Software UART
12 
13 Defined in [softuart.h](@ref softuart.h).
14 
15 The library provides a software UART implementation that uses a single pin for
16 both Tx and Rx (with the help of some additional circuitry). The software and
17 hardware for this design originally came from
18 [this site](http://nerdralph.blogspot.com.au/2014/01/avr-half-duplex-software-uart.html).
19 
20 The library provides a simple method of initialising and reading data from the
21 implemented serial port through the [uartInit()](@ref uartInit), [uartSend()](@ref uartSend)
22 and [uartRecv()](@ref uartRecv) functions. Additional helper functions have been
23 provide to print integers (in both hexadecimal and decimal), strings (from RAM
24 or program memory) and to provide *printf()* style formatted output.
